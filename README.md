## Abstract

A **fully automated end-to-end investment analytics platform** built on a 4-layer stack: Python, MySQL, Power BI, and Excel. The system fetches live market data daily, calculates technical indicators, generates trading signals, stores everything in a cloud MySQL database (Railway), visualizes results in an interactive Power BI dashboard, and auto-generates professional Excel reports — all without manual intervention.

**Key features:**
- Live market data pipeline via Yahoo Finance API
- Cloud MySQL database hosted on Railway
- Interactive Power BI dashboard with real-time filtering
- Auto-generated professional Excel reports (4 sheets)
- Full signal history tracked over time
- Every pipeline run logged automatically

## Architecture

Yahoo Finance (Live Data)
         ↓
    Python
    ├── Fetch live prices
    ├── Calculate RSI / Momentum / MA50 / MA200
    ├── Generate BUY / SELL / HOLD signals
    └── Push everything to MySQL
         ↓
    MySQL Database (Railway Cloud)
    ├── prices     → daily OHLCV + indicators
    ├── signals    → BUY/SELL/HOLD per ticker
    ├── portfolio  → holdings & P&L tracking
    └── run_logs   → every pipeline run logged
         ↓
    Power BI Dashboard
    ├── RSI bar chart per ticker
    ├── Signal distribution pie chart
    ├── Interactive ticker slicer
    └── Price / RSI / Momentum KPI cards
         ↓
    Excel Report 
    ├── Sheet 1 — Summary KPIs
    ├── Sheet 2 — Price History
    ├── Sheet 3 — Signal Log
    └── Sheet 4 — Run Logs


## Database Schema

| Table | Description |
|---|---|
| `prices` | Daily price, MA50, MA200, RSI, Momentum per ticker |
| `signals` | BUY/SELL/HOLD signal with context per ticker per day |
| `portfolio` | Holdings, buy/sell dates, P&L tracking |
| `run_logs` | Timestamp, status, and notes for every pipeline run |
